# chat-app
A real-time chat app made with Django
### First:
Install
```
pip install Django
```
Or
```
pip3 install Django
```
### Second: 
Clone the repository
Inside the project folder, Run the following command in the terminal:
```
python manage.py runserver
```
or
```
python3 manage.py runserver
```
### Finally
Run the local host:
```
http://127.0.0.1:8000/
```
